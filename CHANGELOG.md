# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-05-29

### Added

- CLI entry point with `init`, `config get/set`, `generate` commands
- AI-powered commit message generation from staged git diffs
- Conventional Commits style output
- Interactive config setup (`comet init`)
- JSON output mode for scripting (`--json`)
- API key masking and secure file permissions
- TypeScript type checking via Zod schema validation
